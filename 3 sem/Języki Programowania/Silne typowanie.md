# Równoważność typów
Co to znaczy "ten sam typ"? Silnie typować można na dwa sposoby:
1. Składniowa (nazw). Tylko deklaracja. int == int, != double, float, string.
2. Strukturalna. Jeśli są tak samo reprezentowane (2 bajty to 2 bajty).
