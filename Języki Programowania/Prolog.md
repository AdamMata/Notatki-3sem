# Składnia

| Operator | Prolog |
| -------- | ------ |
| if       | `:-`   |
| not      | `NOT`  |
| and      | `,`    |
| or       | `;`    |

|                | Prolog |
| -------------- | ------ |
| koniec linijki | `.`    |
| zapytanie      | `?-`   |
## Naming convention
`camelCase` lub `snake_case`, ale w `' '` można zapisać dowolną nazwę zmiennej (Atomu)
